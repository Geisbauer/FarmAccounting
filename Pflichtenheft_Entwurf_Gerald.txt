1. Zielstimmungen:
FarmAcoounting (im folgenden kurz "FA" bezeichnet) ist ein Computerprogramm für die Verwaltung des Tierbestandes eines Bauernhofs.
FA ist nicht dafür geeignet, die Ansprüche der Grundsätze ordnungsmäßiger Buchführung (GoB) zu erfüllen. 

2. Einsatz:
FA ist für den einzelnen Benutzer konzipiert, im folgenden "Landwirt" genannt.
Der Landwirt kann mit einer Installation von FA nur einen Tierbestand verwalten.

3. Umgebung:
FA kann auf einem IBM-kompatiblen PC mit dem Microsoft Windows Betriebssystem ab Version Windows 7 installiert werden. 
Der Datenbestand von FA wird in eine ASCII-konforme Textdatei gespeichert und wird nach dem Start des Programms aus dieser ausgelesen.
Es obliegt dem Landwirten, eine Backup-Strategie für diese Textdatei zu implementieren.

4. Funktionalität:
FA bietet folgende Funktionalität:
Für den Schweinebauern:
	Ein Schwein kaufen mit Kaufdatum und Kaufpreis.
	Schwein verkaufen mit Verkaufspreis udn Verkaufsdatum.
Für den Rinderbauern:
	...
Für den Hühnerbauern:
	...
Für alle Bauern:
	...

5. Daten:
Alle Daten müssen vom Landwirt eingeben werden.

6. Leistungen:
Das ausführbare Computerprogramme ("FarmAcounting.exe") wird mit Diskette ausgeliefert und muss vom Landwirt auf den Computer kopiert werden.
Eine Installation ist nicht notwendig.
Beim Kauf von FA ist ein einjähriger Erneuerungsdienst miteingeschlossen. Die Erneuerungen müssen angefordert werden.
Support (vorort oder per Telefon, Telefax, Telex und E-Mail) kann gesondert vereinbart werden.

7. Benutzeroberfläche:
FA wird in einer zeichenorientierten Benutzerschnittstelle (ein Terminal wie z.B. DEC VT100 oder einer entsprechenden Emulation) bedient.

8. Qualität:
Das Nutzungserlebnis von FA wird laufend verbessert.
